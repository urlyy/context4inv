===============================================================================
 PHASE 3: ESBMC/IBMC CONVERSION
===============================================================================

PHASE DESCRIPTION
-----------------
Converts raw .c files to ESBMC-compatible _ibmc.c format. Transforms ACSL
assertions into __VERIFIER_assert calls, converts ∀ quantifiers to
__forall() calls, transforms uninitialized int variable declarations to
nondet_int() assignments, and adds ESBMC headers with FORALL/EXISTS macros.

All 227 _ibmc.c files exist and are generated by convert_to_ibmc.py (buggy).

CRITICAL: _ibmc.c entry point is STILL main() — the converter does NOT
rename main() to foo() and does NOT add a stub main(). This differs from
_frama.c (Phase 2) which uses foo().

No structs or custom types. All operations on primitive int.

TRANSFORMATIONS PERFORMED
-------------------------
1. Header injection: add ESBMC headers including __VERIFIER_assert(),
   __VERIFIER_assume(), __VERIFIER_nondet_int(), __forall(), __exists(),
   FORALL/EXISTS macros, abs(), reach_error().

2. Variable transformation:
   Before:  int x;
   After:   int x = __VERIFIER_nondet_int();
   NOTE: Only applies to scalar int declarations WITHOUT initialization
   and WITHOUT array brackets ([]). Arrays, initialized vars, pointers,
   and multi-decl lines with mixed types are SKIPPED.

3. Assume conversion:
   //@ assume(cond) → __VERIFIER_assume(cond)

4. Assert conversion:
   //@ assert(∀j(cond1 => cond2)) →
   __VERIFIER_assert(__forall((void*)(&j), (!(cond1) || (cond2))))

5. Implication encoding:
   => is converted to || ! via regex substitution.
   KNOWN BUG: This is wrong for nested implications like (A=>B)=>C.

6. FORALL/EXISTS macro definitions (buggy):
   #define FORALL(Var, Type, Cond)       \\\n  Type Var;  ...
   Python string escaping produces broken single-line macros.
   KNOWN BUG.

ENTRY POINT SIGNATURES (Phase 3 — SAME AS Phase 1 raw .c)
-----------------------------------------------------------
  int main (int *a)         -- entry point (NOT renamed to foo())
  int main (int *a, int *b)
  int main (int *a, int *b, int *c)
  int main (int *a, int n)
  int main (void)
  int main (int N, int *a, int *b)
  int main (int *a, int n, int *b, int m)

No stub main() exists. All 227 _ibmc.c files use main() directly.

VERIFIER FUNCTIONS (provided by injected headers)
--------------------------------------------------
  void __VERIFIER_assert(int cond);
    -- aborts with reach_error() if cond is false

  void __VERIFIER_assume(int cond);
    -- constrains nondet values (aborts if cond is false)

  int __VERIFIER_nondet_int();
    -- returns an arbitrary int value (symbolic input)

  void* __forall(void* var, int cond);
    -- ESBMC built-in: asserts cond holds for all values of var

  void* __exists(void* var, int cond);
    -- ESBMC built-in: asserts cond holds for some value of var

  int abs(int x);
    -- absolute value helper (provided inline)

  FORALL(Var, Type, Cond) / EXISTS(Var, Type, Cond)
    -- macros wrapping __invariant(__forall(...))
    -- KNOWN BUG: macros are broken single-line due to Python escaping

EXAMPLE CONVERSION
------------------
Raw .c:
  int main (int *a) {
    int N = 100000;
    int i = 0;
    while (i < N) { a[i] = 42; i = i + 1; }
    //@ assert( ∀x((0 <= x && x < N) => (a[x] == 42)) );
    return 0;
  }

_ibmc.c:
  [ESBMC headers...]
  int main (int *a) {   // NOTE: main(), NOT foo()
    int N = 100000;
    int i = 0;
    while (i < N) { a[i] = 42; i = i + 1; }
    __VERIFIER_assert(__forall((void*)(&x),
      (!(0 <= x && x < N) || (a[x] == 42))));
    return 0;
  }
  // No stub main() — main() IS the entry point

KNOWN BUGS / LIMITATIONS (from convert_to_ibmc.py)
---------------------------------------------------
1. Only handles //@ ACSL (NOT /*@ */ blocks or _frama.c \forall format).
2. int x; → int x = __VERIFIER_nondet_int() for SCALARS only
   (arrays, initialized vars, pointers are SKIPPED).
3. => → || ! via regex — WRONG for nested expressions like (A=>B)=>C.
4. FORALL/EXISTS macro defs produce BROKEN single-line macros.
5. Inline check_nested_loops() is unreliable (use standalone script).
6. Does NOT rename main() → foo() (entry stays as main()).

INVARIANTS FOR PHASE 3
-----------------------
1. Entry function is main() — NOT renamed to foo(). (CRITICAL)
2. No stub main() is appended.
3. No ACSL syntax remains — all assertions are C function calls.
4. Unicode ∀ is replaced by __forall() calls (NOT \forall).
5. Implications are encoded as || ! (disjunction with negated antecedent).
6. ESBMC headers are prepended; the original program body is preserved.
